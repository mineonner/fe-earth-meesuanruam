คะแนนการประเมินความสามารถในการดาเนินงานรายมาตรการและรายตัวชี้วัด

ให้ผู้ประเมินทำเครื่องหมาย หน้าข้อความทุกข้อที่พบเห็นการดำเนินการในเชิงประจักษ์ ทั้งนี้ คะแนนความสามารถในการดำเนินการตามมาตรการแต่ละมาตรการจะ “มาก” หรือ “น้อย” เป็นไปตามจำนวนข้อที่องค์กรปกครองส่วนท้องถิ่นสามารถดำเนินการได้ พร้อมมีเอกสารหรือหลักฐานที่แสดงให้เป็นถึงการดำเนินการ
